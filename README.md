UnityDxrTest
============

This is a repository containing test scenes for
[Unity's realtime ray tracing features](https://unity.com/ray-tracing).

![screenshot](https://i.imgur.com/0aPO5qUl.jpg)

System Requirements
-------------------

- Unity 2022.2
- Windows PC with a DXR compatible graphics card
